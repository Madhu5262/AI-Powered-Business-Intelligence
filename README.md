# AI-Powered Business Intelligence

## Project Overview
AI-Powered Business Intelligence is a data analytics and machine learning project that helps businesses analyze sales data, visualize trends, and predict future sales using Python.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Model Training
- Sales Prediction
- Business Insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib
- Streamlit

## How to Run

1. Install the required libraries:

pip install -r requirements.txt

2. Run the data cleaning script:

python src/data_cleaning.py

3. Run the EDA script:

python src/eda.py

4. Run the visualization script:

python src/visualization.py

5. Train the model:

python src/train_model.py

6. Predict sales:

python src/prediction.py
